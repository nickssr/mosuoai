# OpenClaw

## 简介

OpenClaw 是一个开源的 AI Agent 开发框架，旨在简化智能体应用的构建过程。

## 核心特性

- **多平台支持** - 飞书、微信、Telegram 等
- **Skill 系统** - 灵活的工具调用机制
- **记忆系统** - 长期/会话/工作三层记忆
- **主动性** - Agent 可以主动提醒和跟进

## 快速开始

```bash
npm install -g openclaw
openclaw init my-agent
```
