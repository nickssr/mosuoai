# Claude Code

## 简介

Claude Code 是 Anthropic 推出的命令行 AI 助手，可以直接在终端中执行复杂的开发任务。

## 核心能力

- **代码库导航** - 快速定位和分析代码
- **自动化任务** - 执行重构、测试等任务
- **记忆系统** - 记住用户偏好和项目背景
- **多步骤任务** - autonomously 完成复杂工作流

## 适用人群

- 专业软件开发者
- 技术团队负责人
- 开源项目维护者
