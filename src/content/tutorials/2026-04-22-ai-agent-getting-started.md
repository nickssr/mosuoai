## 什么是 AI Agent？

AI Agent 是能够自主执行任务的 AI 程序，它可以理解用户意图、调用工具、执行操作。

## 环境准备

### 1. 安装 Node.js

确保已安装 Node.js 18 或更高版本。

### 2. 安装 OpenClaw

```bash
npm install -g openclaw
```

## 创建第一个 Agent

### 1. 初始化项目

```bash
openclaw init my-agent
cd my-agent
```

### 2. 配置 Agent

编辑 `config.yaml` 文件，设置 Agent 的基本信息。

### 3. 启动 Agent

```bash
openclaw start
```
