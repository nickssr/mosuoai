## 前言

当 AI Agent 开始替代她手下最优秀的员工时，Clara Shih 决定不能再袖手旁观。这位曾在 Meta 和 Salesforce 领导 AI 业务的高管，如今将全部精力投入一个新使命：**帮助 Z 世代在 AI 取代他们之前，先学会驾驭 AI Agent。**

## 亲历 Agent 替代：震撼的 20 年AI 老兵

Shih 在 AI 领域深耕 20 年，亲眼见证了技术如何一步步重塑职场。但当她看到 AI Agent 在她手下完成原本需要顶级员工才能执行的任务时，她形容自己「感到被激进化了」（You feel radicalized）。

这不是危言耸听。Salesforce、Meta 等科技公司的实践已经证明：单个 AI Agent 可以承担过去需要一个团队才能完成的客户咨询、数据分析、报告生成等工作。而成本不足人类的十分之一。

## New Work Foundation：不做悲观的预言者

面对这一趋势，许多人选择发出警告然后止步。Shih 选择了另一条路——她创立了 **New Work Foundation**，并推出面向消费者的品牌 **Dear CC**，核心目标有两个：

1. **教会 Z 世代使用 AI Agent**：不是简单教 ChatGPT 提问，而是系统学习如何训练、评估、迭代自己的 AI Agent 工作流
2. **培养 Agent 开发能力**：从使用者进阶为构建者，掌握定制化 Agent 的技能

## 为什么是现在？

Shih 指出，Z 世代正处于一个独特的时间节点：他们即将进入职场时，入门级岗位正在被 AI Agent 大规模接管。传统「先做基础工作再逐步晋升」的职业路径正在消失。

但与此同时，**能驾驭 AI Agent 的人才极度稀缺**。Shih 相信，这个供需缺口正是 Z 世代的机会——与其和 AI 竞争同一批岗位，不如成为能指挥 AI 团队的人。

## 争议与质疑

当然，这一倡议也面临质疑。批评者认为，仅靠培训无法解决结构性失业问题——当 60% 的入门级岗位消失时，再多的技能培训也只是杯水车薪。

Shih 承认这一点，但她认为「在系统性变革到来之前，个人能做的就是尽可能占据有利位置」。New Work Foundation 的定位不是乌托邦式的解决方案，而是务实的应对策略。

## 对 AI Agent 开发者的启示

这一事件从侧面印证了 AI Agent 赛道的一个巨大机会：**企业需要大量懂得 Agent 调教的人才**。随着 Agent 渗透到各行各业，如何让人类与 Agent 高效协作、如何设计人机分工流程，将成为新的热门技能方向。

对于 AI Agent 开发者而言，这意味着面向企业客户提供 Agent 培训、咨询、定制服务，可能是一个被低估的商业机会。
