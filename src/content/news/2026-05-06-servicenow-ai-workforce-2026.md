## 前言

5月5日，企业软件巨头 ServiceNow 在拉斯维加斯 Knowledge 2026 大会上抛出了一系列重磅炸弹。这家估值约950亿美元的公司，明确提出了一个核心论点：**AI 作为「助手」的时代结束了，AI 作为「工作者」的时代开始了。**

这不是营销话术。23 million employees——每月通过 ServiceNow 员工门户工作的用户数——加上每年40 million+ cases 的处理量，让这个宣言有了足够的底气。

## 发生了什么

### 全职能AI专家团队上线

ServiceNow 宣布扩展其 Autonomous Workforce（自主工作团队），推出一整套 AI「专家」（AI Specialists），覆盖以下职能领域：

- IT运维
- 客户关系管理（CRM）
- 人力资源（HR）
- 财务
- 法务
- 采购
- 安全与风险管理

这些 AI 专家不是简单的任务助手，而是能**独立完成整个业务流程**的存在——从接收请求到交付结果，全流程无需人工干预，且留下完整的审计追踪记录。

几个早期客户数据值得关注：

| 客户 | 效果 |
|------|------|
| ServiceNow 内部 | IT服务台案件处理速度提升 **99%** |
| DocuSign | 目标实现 **90%** IT工单自主解决 |
| Honeywell | 大部分服务台对话已被 AI 消除 |
| 罗利市政府 | 员工请求偏转率达 **98%**，节省约一个月人力当量 |

### AI Control Tower：治理能力变成「标配」

去年 ServiceOne 推出了 AI Control Tower，用于发现和治理企业内的 AI Agent。今年的更新更具深远意义：**这些能力现在内置于平台的所有产品和套餐中**，不再作为附加功能单独收费。

核心能力包括：

- **持续发现**：自动识别网络中出现的每一个 AI Agent
- **风险评分**：基于行为模式对 Agent 进行风险评级
- **最小权限执行**：强制执行最小权限访问原则
- **业务影响测量**：量化 Agent 对治理标准的实际影响

此外，ServiceNow 还深化了与 Microsoft 的合作，将 AI Control Tower 治理能力扩展到 Microsoft Agent 365 生态系统。这意味着企业可以在统一界面下监控跨两个平台的 AI Agent 行为。

### 安保业务破10亿美元，新增Autonomous Security & Risk

ServiceNow 的安全与风险管理业务去年突破了 **10亿美元** 年合同价值。这次大会上，公司推出了名为 Autonomous Security & Risk 的新产品，整合了两个近期收购：

- **Armis**：跨 IT、OT、IoT、联网设备的持续资产情报
- **Veza**：实时映射企业内所有人和非人类身份及权限

客户数据：一家全球能源公司在70个国家运营，安全威胁遏制时间缩短 **97%**，节省120万工时；一家美国大型金融机构清除了 **96%** 的休眠非人类身份。

### 与 Nvidia 的合作深化

ServiceNow 宣布扩大与 Nvidia 的合作，将 Nvidia 的加速计算基础设施集成到 ServiceNow AI 平台中，目的是帮助企业在规模部署 AI Agent 时获得更高的计算效率和更低的推理成本。

### ServiceNow University：免费学习平台达200万学员

ServiceNow 的免费学习平台 ServiceNow University 推出一年后，学员数已接近 **200万**（同比增长80%）。两个新工具：

- **AI Learning Guide**：对话式学习辅导，根据个人情况构建学习路径
- **SimStudio**：模拟真实 ServiceNow 操作环境的实践平台

## 我们的判断

**这则消息对以下人群有直接影响：**

**企业IT/数字化负责人**：ServiceNow 将 AI Control Tower 变成标配，意味着 AI Agent 治理不再是「选配加钱」的功能，而是平台基础能力。如果你的企业已经在用 ServiceNow，现在有了一个免费的全局 AI Agent 可视化工具。

**HR和业务流程负责人**：AI 专家不再只帮你查资料，而是独立完成招聘流程、员工关系处理等完整工作。需要重新评估哪些岗位的职责边界需要调整。

**安全合规团队**：随着非人类身份（AI Agent）在企业内部的激增，传统的身份管理方案已经不够用了。Armis+Veza 的组合第一次把「谁在用什么系统干什么」这件事说清楚。对于金融、医疗等强监管行业，这是一个值得关注的信号。

**AI Agent 开发者**：ServiceNow 的 Autonomous Workforce 本质上是一套企业级 Agent 编排框架。它的设计思路——角色化、治理内嵌、审计追踪——代表了企业级 Agent 的主流方向。无论你是否使用 ServiceNow，都值得研究它的架构思路。
