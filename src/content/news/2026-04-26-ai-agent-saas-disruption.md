## 前言

AI Agent（智能体）正在以一种前所未有的方式重塑企业软件格局。据 Forbes 报道，AI Agent 的演示已经导致 SaaS（软件即服务）市场市值蒸发约 **2850 亿美元**。这一数字背后，揭示的是整个软件行业商业模式的结构性危机。

## 按席位收费模式的黄昏

传统 SaaS 的核心商业模式是「按席位收费」——每个用户、每个坐席每年支付固定订阅费用。然而，当 AI Agent 可以代表用户自主完成多步骤业务流程时，这种模式的根基开始动摇。

以一个典型知识工作场景为例：过去需要 5 个不同系统账号、多人协作完成的工作流，现在一个 AI Agent 就能端到端执行。这意味着：

- **单一 Agent 替代多个席位**：企业不再需要为每个功能模块购买独立席位
- **订阅费用压缩**：按人头计费的模式受到直接挑战
- **软件采购逻辑重构**：从「给人用」转向「给 Agent 用」

## 市场反应：SaaS 板块持续承压

相关分析指出，iShares 扩展科技-软件 ETF（IGV）今年已下跌近 20%，反映了投资者对 SaaS 前景的深度担忧。Oracle、Meta、Microsoft 等巨头相继宣布裁员，同时加大对 AI 的投入，这一「一减一增」的趋势正在加速。

## 谁是赢家？谁是输家？

Forbes 的分析框架将当下的 AI Agent 浪潮类比为主流计算平台的范式转移。在这场迁移中：

**受益方**：
- AI Agent 平台和基础设施层
- 能够快速适配 Agent 工作流的企业软件

**受损方**：
- 高度依赖席位收费的传统 SaaS 厂商
- 未能及时转型 Agent 能力的软件公司

## 启示

对于 AI Agent 领域的开发者和技术决策者而言，这一趋势意味着：企业软件的采购逻辑正在从「工具思维」转向「能力思维」。未来值得关注的不是软件功能有多强大，而是 Agent 能否自主完成业务目标。
