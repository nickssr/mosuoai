## 核心亮点

微软正式发布 AutoGen v3.0，多 Agent 编排框架迎来重大更新。

## 主要特性

### 1. 灵活编排

支持更复杂的 Agent 协作模式，包括分层、并行、串行等。

### 2. 对话管理

改进多轮对话管理，支持上下文切换和话题恢复。

### 3. 任务分配

智能任务分配算法，根据 Agent 能力自动分配任务。

## 开发者指南

AutoGen v3.0 提供完整的 SDK 和文档，开发者可快速上手构建多 Agent 系统。
